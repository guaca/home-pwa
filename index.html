Vamos a jugar a un juego...

En esta web hay creados 3 subdominios distintos

En cada subdominio se ha creado un proyecto con Nuxt.js junto con el módulo PWA, pero cada uno configurado de un modo distinto para generar la aplicación correspondiente.

Aquí tenéis los enlaces a cada una de las opciones:

SSR Static SPA

¿Qué significa todo esto?

SSR (Server Side Rendered)

En el primer caso, utilizamos Nuxt.js como framework para renderizar la aplicación del proyecto.

Static Generated

En este caso, Nuxt.js genera un HTML para cada una de las rutas y lo guarda en un archivo. Es decir, en este caso hace un pre rendering del proyecto.

SPA (Single Pag Application)

Por último, Nuxt.js ofrece también la opción de generar una SPA, utilizando Nuxt.js como si trabajásemos con Vue.js.
